## Inclusion in IM projects

Add this entry in `composer.json` :

```
"repositories": [
        {
            "type": "vcs",
            "url": "git@github.com:infernalmedia/laravel-generator.git"
        }
    ],
```

Then, run `composer require infyomlabs/laravel-generator`
